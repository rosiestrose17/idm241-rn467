Slideshows

Triggers
User Interaction : When the user clicks on navigation arrows on the left or right
Automatic Timer : If the user doesn't interact with the slideshow, it will trigger specific actions after a set period of time

Rules
Slide Transitions: the slide transition involves sliding in from left to right or right to left.
Navigation Controls: Clicking the right arrow should move to the next slide, and clicking the left arrow should go to the previous slide.
Pause Automatic Timer: the slideshow pauses when the user hovers over a drink

Feedbacks
Visual Feedback :the button changes from orange to blue when hovered, and adds underline when clicked
Automatic Feedback: the drink slides from right to left one by one when there is no interaction from the user

Loops & Modes
Looping : The drinks cycle back to the first one once they reach the end.
Mode Selection: Manual mode (where users control slide transitions), and automatic mode (where slides change automatically).

Hovering Interaction

Triggers
When the user's mouse pointer enters the image's (a drink) area, the interaction is initiated.

Rules
Scale Up on Hover : When the user's mouse hovers over the image, a rule is applied that instructs the image to scale up slightly.
Scale Back on Mouse Leave: When the user moves their mouse away from the image (mouse leaves), the image returns to its original size gradually.

Feedbacks
Visual Feedback : When the user hovers over the image and it scales up, when the mouse leaves, and the image scales back down.

Pop-up Description

Triggers
Click on item : When the user clicks on an image of a drink
Click on Arrows : User clicks on the left and right arrow buttons within the pop-up.
Close pop-up: User clicks on close button or areas outside the pop-up.

Rules
Show Pop-up: When the user clicks on an image, the pop-up description associated with that image will become visible.
Hide Pop-up: When the user clicks outside the pop-up or clicks a close button within the pop-up., the description should be hidden or closed.
Change Pop-up Content: When the user clicks the left or right arrow, the pop-up content (image and description) changes to the previous or next item in the sequence.

Feedbacks
Visual Feedback (Display): When the user clicks on the image, the pop-up description appears on the center screen and the background darkens.
Visual Feedback (Close): The close button changes from pink to white when hovers or clicked.
Visual Feedback (Arrows): When the user hovers over or clicks the arrow buttons, the arrows change from pink to white
Visual Feedback (Pop-up): When the user clicks an arrow button, the pop-up content changes instantly.

Loops & Modes
Single Instance: Clicking on one image will display the description for that image only.
Repeatable: Users can click on any image to reveal its respective pop-up description.
Sequential Loop: The arrows create a loop within the pop-up, allowing users to cycle through the items sequentially. Clicking the right arrow advances to the next item, and clicking the left arrow goes to the previous item.
Infinite Loop: When the user reaches the beginning or end of the sequence, the pop-up content wraps around to the opposite end, providing continuous navigation.
