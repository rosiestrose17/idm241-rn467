Triggers
User Interaction : When the user clicks on navigation arrows or dots, it starts the animation to move to the next or previous slide. Additionally, hovering over the carousel may trigger elements like pause/play buttons or slide previews.
Automatic Timer : Another trigger is an automatic timer. If the user doesn't interact with the carousel, it automatically progresses to the next slide after a predefined time interval, typically seen in auto-playing carousels.

Rules
Slide Transitions : How the slides transition from one to another: sliding from left to right, and fading in and out.
Navigation Controls: Clicking the right arrow should move to the next slide, and clicking the left arrow should go to the previous slide. Clicking on a navigation dot should take the user directly to the corresponding slide.

Feedbacks
Visual Feedback : Provide visual feedback to the user during interaction. Highlight the currently active slide, change the appearance of navigation dots to indicate the current slide, and possibly use animations or transitions to make these changes more engaging
User-Controlled Feedback: include pause/play buttons so users can control the automatic timer. When they interact with navigation elements, provide visual cues like button depressions or hover effects to acknowledge their actions.

Loops Modes
Infinite Loop : If the user reaches the last slide, it loops back to the first slide.
