Alpha Build

Triggers
User Interaction: When the user's mouse pointer enters the image's (a drink) area, the interaction is initiated.
Rules
Scale Up on Hover: When the user's mouse hovers over the image, the image scales up about 10% over time.
Scale Back on Mouse Leave: When the user moves their mouse away from the image (mouse leaves), the image returns to its original size over time.
Feedbacks
Scale Up on Hover: When the user's mouse hovers over the image, the image scales up about 10% over time.
Scale Back on Mouse Leave: When the user moves their mouse away from the image (mouse leaves), the image returns to its original size over time.
Loops & Modes
Looping: This interaction does not involve looping; it occurs only once when the user hovers over the item and again when they hover off.
Mode Selection: The user manually initiates the scaling effect by hovering over the item, which is a user-controlled mode. When the user hovers off, the image automatically returns to its original size, which is an automatic mode triggered by the user's action.
