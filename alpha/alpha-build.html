Alpha Build

Triggers
User Interaction: When the user's mouse pointer enters the image's (a drink) area, the interaction is initiated.
Rules
Scale Up on Hover: When the user's mouse hovers over the image, the image scales up about 10% over time.
Scale Back off Hover: When the user's mouse hovers off the image, the image returns to its original size over time.
Cursor Change : When the user hovers their mouse pointer over the image, the cursor changes to a hand pointer icon. When the user hovers the mouse pointer off the image, it reverts to the default arrow cursor.
Feedbacks
Scale Up on Hover: When the user's mouse hovers over the image, the image scales up about 10% over time.
Scale Back off Hover: When the user's mouse hovers off the image, the image returns to its original size over time.
Cursor Change : When the user hovers their mouse pointer over the image, the cursor changes to a hand pointer icon. When the user hovers the mouse pointer off the image, it reverts to the default arrow cursor.
Loops & Modes
Looping: This interaction does not involve looping; it occurs only once when the user hovers over the item and again when they hover off.
